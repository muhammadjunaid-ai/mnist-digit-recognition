# MNIST Digit Recognition App

This project is a Deep Learning web app built using TensorFlow and Streamlit.

Users can draw a digit (0–9) and the model predicts the number using a CNN trained on the MNIST dataset.

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- PIL

## How to Run

Install requirements:

pip install -r requirements.txt

Run the app:

streamlit run app.py
